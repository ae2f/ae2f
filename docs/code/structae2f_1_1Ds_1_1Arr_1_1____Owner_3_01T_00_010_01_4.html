Detailed Description

template<typename T>
struct ae2f::Ds::Arr::__Owner< T, 0 >
Warning
Using this class directly may cause undefined behaviour.
Using the binder instead is recommended.

This structure owns the memory.

Suggest the type is no trivial.
Therefore the element type will be ae2f::Ds::Arr::Pack<T>, in order to avoid the data destruction under wrong responsibility.
