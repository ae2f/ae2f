template<typename T>
struct ae2f::Ds::Arr::__Refer< T, 1 >
Warning
Using this class directly may cause undefined behaviour.
Using the binder instead is recommended.

This structure does not own the memory.

Suggest the type is trivial.
Therefore the element type will be [T] itself.
